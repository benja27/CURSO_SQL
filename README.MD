* Data manipulation Languaje (DML): SELECT, INSERT, UPDATE, DELETE 
* Data Control Languaje (DCL): GRANT, REVOKE 
* Transaction Control Languaje (TCL): COMMIT, ROLLBACK, SET TRANSACTION
